# CircuitBuilder
Here is my project on AC circuits. To run the code, put the following command in the terminal: 'g++ -g -std=c++17 main.cpp circuit.cpp complex.cpp component.cpp inductor.cpp capacitor.cpp resistor.cpp -o circuit_program', followed by the command: './circuit_program'  The code is fairly easy to follow, the program will ask how many components the user would like to put in the circuit and then ask the user to place the first component. Then the user can add components at their own leisure and will have the option to put more components in the ciruits once the circuit component library is empty.  Note that to find a component, the key is the component type with its specific count with no spaces in between, and the key will be clear when the component is initialised when the library is set and the program will say "Choosing data for: (key)".
